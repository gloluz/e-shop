# e-shop

## Description

This project is an API to manage online shop.

## Installation

First, clone the project :

```
git clone git@github.com:gloluz/e-shop.git
```

Then you must install the project :

```
npm ci
```

## Comands

### Start the server

```
npm start
```

### To format code

```
npm run format
```
